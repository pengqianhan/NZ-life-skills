# Skills Index

This page indexes the skills in `NZ-life-skills` for humans, search engines, and agents.

## Audience

This repository is designed for:

- international students arriving in New Zealand
- Chinese newcomers who want bilingual guidance
- Codex users
- Claude Code users
- maintainers building agent-friendly onboarding skills

## Skill Categories

### Identity And Access

#### `kiwi-access-card`

- Purpose: Explain Kiwi Access Card eligibility, online versus in-person application, required documents, fees, timelines, and PDF form handling.
- Typical questions:
  - Can an international student apply for a Kiwi Access Card?
  - Can I apply online with a foreign passport?
  - Do I need to print the application form?
- Files:
  - [SKILL.md](../skills/kiwi-access-card/SKILL.md)
  - [Current policy](../skills/kiwi-access-card/references/current-policy.md)

## Repository Rules
