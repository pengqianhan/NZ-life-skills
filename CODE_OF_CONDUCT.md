# Code of Conduct

## Our Standard

Contributors should help keep this repository useful, respectful, and practical.

Expected behavior:

- be respectful and direct
- focus on improving the repository
- cite official sources when discussing policy-sensitive topics
- explain disagreements in technical or factual terms

Unacceptable behavior:

- harassment or personal attacks
- discriminatory language
- deliberate misinformation
- hostile or disruptive conduct

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project interactions.

## Enforcement

Project maintainers may remove, edit, or reject contributions and discussions that do not follow this code of conduct.

## Reporting

If needed, open a private communication channel with the maintainer through GitHub.
