# NZ IRD Number

## Overview

Use this skill to guide a newly arrived student through IRD number setup without overstating stale details. Treat IRD requirements as policy-sensitive and verify official Inland Revenue sources before giving operational instructions.

## Workflow

1. Determine whether the user needs:
   - first-time IRD application
   - document checklist
   - help with bank or employer linkage
   - tax code explanation
   - clarification about current processing steps
2. Read [references/current-guidance.md](references/current-guidance.md).
3. For any current process detail, official web verification is required before answering.
4. If live guidance differs from the reference, update the reference file.
5. Explain the process in plain language and separate:
   - identity and residency evidence
   - tax administration steps
   - practical follow-up steps after the number is issued

## Source Order

1. Inland Revenue official pages and tools
2. Official IRD forms or document checklists
3. New Zealand government guidance that links back to Inland Revenue

Do not answer current tax-administration questions from memory when the official site is available.

## Answering Rules

- State whether the user likely needs an IRD number based on the scenario they describe.
- Separate `what the IRD number is for` from `how to apply`.
- For students, include practical follow-up questions:
  - job onboarding
  - bank account linkage
  - tax code selection
- If the user asks for exact current documents, processing times, or portal steps, verify live official guidance first.
- If a user shares a form or screenshot, read the artifact directly instead of guessing from old process knowledge.

## References

- Read [references/current-guidance.md](references/current-guidance.md) before answering.
- Update that file whenever current official rules materially change.
