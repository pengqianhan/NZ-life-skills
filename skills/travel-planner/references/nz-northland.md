# 参考资料：新西兰北地（Northland）路线
**Reference: New Zealand Northland Route**

> 供 `travel-planner` skill 在规划新西兰奥克兰以北行程时使用。
> Read this file when the user is planning a trip in New Zealand's Northland / Bay of Islands region.

---

## 地理概览

奥克兰 → 北地 主干路线（State Highway 1 → SH10）：

```
奥克兰 (Auckland)
  ↓ 45min
Matakana（集市小镇）
  ↓ 45min
Eutopia Cafe（网红咖啡，Kaiwaka附近）
  ↓ 45min
Whangarei（北地最大城市）
  ↓ 35min
Kawakawa（Hundertwasser艺术卫生间）
  ↓ 30min
Paihia（Bay of Islands 旅游中心）
  ↓ 20min
Kerikeri（推荐住宿基地）
  ↓ 2.5h（北上）
Cape Reinga（新西兰最北端）
```

**关键距离：**
- 奥克兰 → Kerikeri: 约 3 小时（无停留）
- 奥克兰 → Cape Reinga: 约 5 小时（无停留）
- Kerikeri → Paihia: 约 20 分钟
- Paihia → Cape Reinga（自驾）: 约 3 小时

---

## 最佳住宿基地

**推荐：Kerikeri**（首选）
- 北地地理中心，距Paihia仅20分钟，距Cape Reinga 2.5小时
- 比Paihia安静，Airbnb性价比更高
- 周边：Stone Store（新西兰最古老石砌建筑）、Rainbow Falls（彩虹瀑布）

**备选：Paihia**
- 海滨城镇，旅游设施齐全，所有游船从此出发
- 旺季（12月-1月）价格贵，旺季期间 Kerikeri 更划算

**预算选择：Whangarei**
- 城市设施完善，距Paihia 1小时，适合预算有限者

## 景点详解

### Cape Reinga（雷恩加角）⭐ 必去
- 新西兰最北端灯塔，太平洋与塔斯曼海在此交汇（可见两海颜色差异）
- 毛利文化圣地（灵魂离开世界之处）
- **强烈推荐参加有组织的一日游**，而非自驾（行程含沙丘、海滩驾车等仅团队提供的体验）

### 可口可乐湖 / 单宁湖（Cola/Tannin Lake）
- 因植被中单宁酸渗入，湖水呈深棕色如可乐
- Cape Reinga 团行程途经，无需单独安排

### Te Paki 大沙丘 ⭐
- Cape Reinga 附近，团队提供沙板，可滑沙
- 约1.5小时活动时间
- 贴士：穿旧衣服，细沙会进入鞋和裤子

### 九十英里海滩（Ninety Mile Beach）
- 实际长88公里，大巴沿沙滩行驶（普通私家车不建议上沙滩）
- 感受大西洋海浪打车轮的特别体验

### Bay of Islands / Paihia
- 144个小岛，水上活动丰富
- 主要出发点：Paihia码头
- 可做：海豚游船、Hole in the Rock游船、帆船、皮划艇、Russell渡轮

### Waitangi Treaty Grounds（怀唐义条约地）
- 新西兰建国历史核心地点，1840年英国与毛利酋长签约处
- 含毛利战船、会堂、博物馆，体验约2-3小时
- 门票约 NZD $35/人（2026年参考价）

### Matakana
- 手工食品集市（周六早8点-下午1点）
- 精品酒庄（Ascension Wine Estate）
- 网红咖啡馆集中区
- 奥克兰出发后约45分钟，是很好的第一站

### Eutopia Cafe（Kaiwaka附近）
- 山坡上的网红打卡咖啡馆，自手烘焙咖啡
- 风景美，拍照佳，人流适中（非主流景点）
- 从Matakana继续向北约45分钟

### Kawakawa
- 著名 Hundertwasser 公厕（艺术大师设计的免费公共卫生间）
- 路过必拍，停车5-10分钟即可
