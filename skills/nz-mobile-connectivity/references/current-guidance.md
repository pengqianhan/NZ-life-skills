## Common Use Cases

- Student lands in New Zealand and needs immediate data for maps and messaging
- User wants the cheapest workable prepaid plan for the first month
- User wants to know whether their phone supports local eSIM activation
- User wants to switch later from a temporary prepaid setup to a better-value long-term plan

## Stable Framing

- Mobile setup is usually one of the first practical tasks after arrival.
- The best choice depends on urgency, phone compatibility, and how much data the student needs.
- Current plan recommendations should always be based on live carrier pages rather than stored assumptions.
