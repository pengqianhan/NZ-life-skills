# NZ Bank Account

## Overview

Use this skill to help a new student understand the banking setup sequence without overcommitting to any one bank's changing requirements. Treat account-opening details as bank-specific and time-sensitive.

## Workflow

1. Determine whether the user needs:
   - which bank to consider
   - what documents are usually needed
   - how to open an account after arrival
   - help linking the account to work, rent, or IRD
2. Read [references/current-guidance.md](references/current-guidance.md).
3. If the user asks for current account fees, current bank offers, branch processes, or exact document lists, verify on official bank sites first.
4. If a specific bank is mentioned, prefer that bank's official source.
5. Answer with the distinction between:
   - general newcomer banking workflow
   - bank-specific current requirements

## Source Order

1. Official pages of the bank the user names
2. General New Zealand consumer or government guidance if needed for context
3. Student support pages only when they link to official bank flows or summarize campus-specific practice

## Answering Rules

- Explain the likely setup sequence:
  - choose bank
  - gather identity and address evidence
  - open account
  - activate card and digital banking
  - link to employer, rent, or transfers
- For students, cover practical pain points:
  - arrival before permanent housing
  - temporary address
  - receiving card by mail
  - using overseas funds in the first week
- Keep product recommendations conditional on current research.

## References

- Read [references/current-guidance.md](references/current-guidance.md) before answering.
